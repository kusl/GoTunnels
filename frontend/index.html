GoTunnels

A fully self-contained demo stack: a Go API and a plain HTML/CSS/JS web app, each reachable only through its own Cloudflare Quick Tunnel, with Postgres and telemetry behind them. Nothing binds to a host port.

What this demonstrates

  • Passkey (WebAuthn) login, password fallback, and optional TOTP two-factor.
  • Server-side sessions with a Bearer token that works cross-origin.
  • Privacy-preserving audit log: IPs are hashed with a per-instance pepper, never stored raw.
  • OpenTelemetry traces, metrics, and logs over OTLP — vendor-neutral.
  • Content-Security-Policy in report-only mode, reported back to the API.
  • No exposed host ports; every service talks over the internal Compose network.

Get started

Create an account, then add a passkey or enable TOTP from settings.
